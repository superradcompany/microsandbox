See [Secrets](/sandboxes/secrets) for how placeholder substitution works and usage examples.

Secrets use placeholder substitution — the real value never enters the VM. The placeholder is swapped in by the TLS proxy only when traffic targets an allowed host.

```typescript
import { Sandbox } from "microsandbox";

// Shorthand: auto-generates placeholder `$MSB_OPENAI_API_KEY`.
await using sb = await Sandbox.builder("agent")
  .image("python")
  .secretEnv("OPENAI_API_KEY", process.env.OPENAI_API_KEY!, "api.openai.com")
  .create();

// Full control via SecretBuilder.
await using sb2 = await Sandbox.builder("agent2")
  .image("python")
  .secret((s) =>
    s
      .env("STRIPE_KEY")
      .value(process.env.STRIPE_KEY!)
      .allowHost("api.stripe.com")
      .allowHostPattern("*.stripe.com")
      .injectHeaders(true)
      .injectQuery(false),
  )
  .create();
```

## SecretBuilder

Builder consumed by [`SandboxBuilder.secret(s => …)`](/sdk/typescript/sandbox#sandboxbuilder) and [`NetworkBuilder.secret(s => …)`](/sdk/typescript/networking#networkbuilder). Every setter returns the builder.

| Method | Signature | Description |
|--------|-----------|-------------|
| env | `(varName: string) => this` | Environment variable to expose the placeholder under. **Required.** |
| value | `(value: string) => this` | The real secret value (stays on the host). **Required.** |
| placeholder | `(placeholder: string) => this` | Custom placeholder string. Defaults to `$MSB_<envVar>`. |
| allowHost | `(host: string) => this` | Allow substitution to a specific host (matched against TLS SNI). |
| allowHostPattern | `(pattern: string) => this` | Allow substitution to any host matching a wildcard pattern. |
| allowAnyHostDangerous | `(iUnderstandTheRisk: boolean) => this` | Permit substitution to any host. Acknowledge the risk explicitly. |
| requireTlsIdentity | `(enabled: boolean) => this` | Require a verified TLS identity before substituting. Default `true`. |
| injectHeaders | `(enabled: boolean) => this` | Allow substitution in HTTP headers. Default `true`. |
| injectBasicAuth | `(enabled: boolean) => this` | Allow substitution in the HTTP Basic Auth credential. Default `true`. |
| injectQuery | `(enabled: boolean) => this` | Allow substitution in URL query parameters. Default `false`. |
| injectBody | `(enabled: boolean) => this` | Allow substitution in the HTTP request body. Default `false`. |
| build | `() => SecretEntry` | Materialize the entry. Throws [`InvalidConfigError`](/sdk/typescript/sandbox#errors) if `env` or `value` are missing. |

## Types

### SecretEntry

The value type produced by [`SecretBuilder.build()`](#secretbuilder) and stored on `SandboxConfig.secrets` and `NetworkConfig.secrets`. All fields are read-only.

| Field | Type | Description |
|-------|------|-------------|
| envVar | `string` | Environment variable name. |
| value | `string` | Secret value (host-only). |
| placeholder | `string \| null` | Placeholder string. `null` means auto-generate as `$MSB_<envVar>`. |
| allowedHosts | `readonly string[]` | Allowed hosts (exact match against TLS SNI). |
| allowedHostPatterns | `readonly string[]` | Wildcard host patterns. |
| allowAnyHost | `boolean` | Whether substitution is permitted to any host. |
| requireTlsIdentity | `boolean` | Whether a verified TLS identity is required. |
| injection | [`SecretInjection`](#secretinjection) | Per-substitution-point flags. |

### SecretInjection

Controls where the TLS proxy substitutes the placeholder with the real value. Each flag is optional; omitted flags use the default.

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| headers | `boolean` | `true` | Replace the placeholder anywhere in the HTTP headers. |
| basicAuth | `boolean` | `true` | Replace the placeholder specifically in `Authorization` Basic Auth. No-op when `headers` is `true`. |
| queryParams | `boolean` | `false` | Replace the placeholder in URL query strings. |
| body | `boolean` | `false` | Replace the placeholder in the HTTP request body. When enabled, `Content-Length` is rewritten. |

### ViolationAction

Action taken when a secret placeholder is sent to a disallowed host. Set via [`NetworkBuilder.onSecretViolation()`](/sdk/typescript/networking#networkbuilder).

| Value | Description |
|-------|-------------|
| `'block'` | Silently drop the request. The guest sees a connection reset. Default. |
| `'block-and-log'` | Drop the request and emit a warning log on the host side. |
| `'block-and-terminate'` | Drop the request, log an error, and shut down the entire sandbox. |
