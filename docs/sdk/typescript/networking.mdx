## NetworkBuilder

Builder used inside [`SandboxBuilder.network(n => â€¦)`](/sdk/typescript/sandbox#sandboxbuilder) to configure ports, policy, DNS, TLS, and network-scoped secrets. Every setter returns the builder.

```typescript
import { Sandbox, NetworkPolicy, Rule, Destination } from "microsandbox";

// Default â€” public internet only (the runtime default).
await using publicOnly = await Sandbox.builder("public").image("alpine").create();

// Fully airgapped.
await using isolated = await Sandbox.builder("isolated")
  .image("alpine")
  .network((n) => n.policy(NetworkPolicy.none()))
  .create();

// Custom rules â€” first match wins, evaluated independently per direction.
await using custom = await Sandbox.builder("custom")
  .image("alpine")
  .network((n) =>
    n.policy({
      defaultEgress: "deny",
      defaultIngress: "allow",
      rules: [
        Rule.allowEgress(Destination.domain("api.openai.com")),
        Rule.denyEgress(Destination.group("metadata")),
      ],
    }),
  )
  .create();

// DNS filtering and TLS interception.
await using filtered = await Sandbox.builder("filtered")
  .image("alpine")
  .network((n) =>
    n
      .dns((d) => d.blockDomain("blocked.example.com").blockDomainSuffix(".evil.com"))
      .tls((t) => t.bypass("*.googleapis.com").interceptedPorts([443])),
  )
  .create();
```

| Method | Signature | Description |
|--------|-----------|-------------|
| enabled | `(enabled: boolean) => this` | Toggle networking. Default `true`. |
| port | `(hostPort: number, guestPort: number) => this` | Publish a TCP port `host â†’ guest`. |
| portUdp | `(hostPort: number, guestPort: number) => this` | Publish a UDP port `host â†’ guest`. |
| policy | `(policy: NetworkPolicy) => this` | Set the rule list and per-direction defaults. |
| dns | `(configure: (b: DnsBuilder) => DnsBuilder) => this` | Configure DNS interception. |
| tls | `(configure: (b: TlsBuilder) => TlsBuilder) => this` | Configure TLS interception. |
| secret | `(configure: (b: SecretBuilder) => SecretBuilder) => this` | Add a network-scoped secret. See [Secrets](/sdk/typescript/secrets). |
| secretEnv | `(envVar: string, value: string, placeholder: string, allowedHost: string) => this` | Shorthand for a single-host secret with explicit placeholder. |
| onSecretViolation | `(action: ViolationAction) => this` | Action when a secret is sent to a disallowed host. |
| maxConnections | `(max: number) => this` | Cap concurrent connections. |
| trustHostCAs | `(enabled: boolean) => this` | Ship the host's trusted CAs into the guest at boot (off by default). |

## NetworkPolicy

Factory + interface â€” `NetworkPolicy` exports preset constructors as well as the underlying value shape.

| Factory | Description |
|---------|-------------|
| `NetworkPolicy.none()` | Deny everything in both directions (fully airgapped). |
| `NetworkPolicy.allowAll()` | Allow everything in both directions. |
| `NetworkPolicy.publicOnly()` | Egress allowed only to public destinations; ingress allowed by default. **Runtime default.** |
| `NetworkPolicy.nonLocal()` | Egress allowed to public + private (LAN); ingress allowed by default. |

### NetworkPolicy (interface)

```typescript
interface NetworkPolicy {
  readonly defaultEgress: Action;    // "allow" | "deny"
  readonly defaultIngress: Action;   // "allow" | "deny"
  readonly rules: readonly Rule[];   // ordered, first-match-wins per direction
}
```

## Rule

Factory + interface for individual policy rules. First-match-wins, evaluated independently for egress and ingress.

| Factory | Description |
|---------|-------------|
| `Rule.allowEgress(destination)` | Allow outbound traffic to `destination`. |
| `Rule.denyEgress(destination)` | Deny outbound traffic to `destination`. |
| `Rule.allowIngress(destination)` | Allow inbound traffic from `destination`. |
| `Rule.denyIngress(destination)` | Deny inbound traffic from `destination`. |
| `Rule.allowAny(destination)` | Allow in both directions. |
| `Rule.denyAny(destination)` | Deny in both directions. |

### Rule (interface)

```typescript
interface Rule {
  readonly direction: Direction;          // "egress" | "ingress" | "any"
  readonly destination: Destination;
  readonly protocols: readonly Protocol[]; // empty = any protocol
  readonly ports: readonly PortRange[];    // empty = any guest port
  readonly action: Action;                 // "allow" | "deny"
}
```

`Protocol` is `"tcp" | "udp" | "icmpv4" | "icmpv6"`. `Direction` is `"egress" | "ingress" | "any"`. `Action` is `"allow" | "deny"`.

## Destination

Factory + discriminated union for rule destinations.

| Factory | Result |
|---------|--------|
| `Destination.any()` | `{ kind: "any" }` |
| `Destination.cidr("10.0.0.0/8")` | `{ kind: "cidr"; cidr }` |
| `Destination.domain("example.com")` | `{ kind: "domain"; domain }` |
| `Destination.domainSuffix(".example.com")` | `{ kind: "domainSuffix"; suffix }` |
| `Destination.group("public")` | `{ kind: "group"; group: DestinationGroup }` |

`DestinationGroup` is `"public" | "loopback" | "private" | "link-local" | "metadata" | "multicast" | "host"`. The host machine is reachable from inside the guest as `host.microsandbox.internal`.

## PortRange

| Factory | Result |
|---------|--------|
| `PortRange.single(port)` | `{ start: port, end: port }` |
| `PortRange.range(start, end)` | `{ start, end }` |

## DnsBuilder

Builder for [`NetworkBuilder.dns(d => â€¦)`](#networkbuilder).

| Method | Signature | Description |
|--------|-----------|-------------|
| blockDomain | `(domain: string) => this` | Block a specific FQDN (returns REFUSED). |
| blockDomainSuffix | `(suffix: string) => this` | Block any name ending in `suffix`. |
| nameservers | `(servers: Iterable<string>) => this` | Override upstream nameservers (`IP`, `IP:PORT`, `HOST`, or `HOST:PORT`). Hostnames are resolved once at startup via the host's resolver. |
| rebindProtection | `(enabled: boolean) => this` | Toggle DNS rebinding protection. |
| queryTimeoutMs | `(ms: number) => this` | Per-query timeout in milliseconds. |

## TlsBuilder

Builder for [`NetworkBuilder.tls(t => â€¦)`](#networkbuilder).

| Method | Signature | Description |
|--------|-----------|-------------|
| bypass | `(pattern: string) => this` | Skip interception for hosts matching this glob (e.g. `"*.internal.corp"`). |
| verifyUpstream | `(verify: boolean) => this` | Whether to verify upstream certificates (default: `true`). |
| interceptedPorts | `(ports: Iterable<number>) => this` | TCP ports to intercept (default `[443]`). |
| blockQuic | `(block: boolean) => this` | Block QUIC on intercepted ports. |
| upstreamCaCert | `(path: string) => this` | PEM file with extra root CAs the proxy should trust. |
| interceptCaCert | `(path: string) => this` | PEM file used as the intercepting CA's certificate. |
| interceptCaKey | `(path: string) => this` | PEM file used as the intercepting CA's private key. |

## Types

### NetworkConfig

Built network configuration produced by `NetworkBuilder.build()` and exposed as [`SandboxConfig.network`](/sdk/typescript/sandbox#sandboxconfig). All fields are read-only.

| Field | Type | Description |
|-------|------|-------------|
| enabled | `boolean` | Whether networking is enabled. |
| ports | `readonly PublishedPort[]` | Published ports (`{ hostPort, guestPort, protocol }`). |
| policy | [`NetworkPolicy`](#networkpolicy) `\| null` | Rule list with per-direction defaults. |
| dns | [`DnsConfig`](#dnsconfig) `\| null` | DNS interception configuration. |
| tls | [`TlsConfig`](#tlsconfig) `\| null` | TLS interception configuration. |
| secrets | `readonly` [`SecretEntry`](/sdk/typescript/secrets#secretentry)`[]` | Network-scoped secrets. |
| secretViolation | [`ViolationAction`](/sdk/typescript/secrets#violationaction) `\| null` | Action when a secret is sent to a disallowed host. |
| maxConnections | `number \| null` | Connection cap. |
| trustHostCAs | `boolean` | Whether to ship host CAs into the guest. |

### DnsConfig

| Field | Type | Description |
|-------|------|-------------|
| blockedDomains | `readonly string[]` | Exact domains to block (REFUSED). |
| blockedSuffixes | `readonly string[]` | Suffix-blocked domains. |
| nameservers | `readonly string[]` | Upstream nameservers. |
| rebindProtection | `boolean \| null` | Rebind protection toggle. |
| queryTimeoutMs | `number \| null` | Per-query timeout. |

### TlsConfig

| Field | Type | Description |
|-------|------|-------------|

| interceptedPorts | `readonly number[]` | TCP ports under interception. |

| upstreamCaCertPaths | `readonly string[]` | Extra trusted root CA PEMs. |

| interceptCaKeyPath | `string \| null` | Custom intercepting CA key path. |

```typescript
interface PublishedPort {
```
