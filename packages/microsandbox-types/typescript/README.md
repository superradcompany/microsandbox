# @microsandbox/types

Shared task and wire contract types for microsandbox, as TypeScript.

This package is the type-only mirror of the `microsandbox-types` Rust crate's sandbox spec building blocks. It gives TypeScript consumers the exact sandbox spec shapes (rootfs sources, mounts, patches, resources, policy, networking sub-documents) the rest of microsandbox uses, so a `RootfsSource` or `VolumeMount` means the same thing on both sides of the wire.

Only the `#[typeshare]`-annotated types are emitted. The cloud wire DTOs (`CloudCreateSandboxRequest`, …) and the top-level container specs (`SandboxSpec`, `NetworkSpec`, `VolumeSpec`, …) are intentionally out of scope for these bindings.

It contains no runtime values beyond the generated `enum`s. Every other export is a type or interface.

## Generated, Not Hand-Written

`src/index.ts` is generated from the Rust crate with [typeshare](https://github.com/1Password/typeshare) — the same codegen that emits the Go bindings — and carries a `Generated by typeshare` header. Do not edit it. To change a shape, edit the Rust type and regenerate from the repo root (requires `cargo install typeshare-cli`):

```bash
just gen
```

## Install

```bash
npm install @microsandbox/types
```

## Usage

Import the shapes you need with `import type` (enums are runtime values, so import those without `type`):

```ts
import type { RootfsSource, SandboxResources } from "@microsandbox/types";

const image: RootfsSource = { type: "oci", content: { reference: "python" } };

const resources: SandboxResources = {
  vcpus: 2,
  memory_mib: 1024,
};
```

## Generated Shape Notes

The bindings follow typeshare conventions, which mirror the Rust serde representation:

- Data-carrying Rust enums become adjacent-tagged unions with snake_case tags. `RootfsSource` is `{ type: "bind", content: string } | { type: "oci", content: OciRootfsSource } | { type: "disk_image", content: {...} }`.
- Unit enums become TypeScript `enum`s. `StatVirtualization` is `enum StatVirtualization { Strict = "strict", Relaxed = "relaxed", Off = "off" }`.
- `VolumeMount`, `Patch`, `Destination`, `HostPattern`, and `ViolationAction` are adjacent-tagged unions on `{ type, content }` with snake_case tags (e.g. `"bind"`, `"disk_image"`, `"block_and_log"`).
- Optional Rust fields are rendered `?:` optional (`upper_size_mib?: number`).
- Host paths, CIDRs, and IP addresses (`PathBuf`, `IpNetwork`, `Ipv4Network`, …) serialize as `string` via `#[typeshare(serialized_as = ...)]`.
